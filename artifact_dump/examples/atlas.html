The full capability map

✓SVG (charts, diagrams, icons, maps)

✓WebGL / 3D (same sandbox, GPU-backed)
✓Web Audio (synths, SFX, no files)
✓Native <dialog>, <details>, forms
✓Drag & drop, Clipboard, pointer events
✓CSS animation, transitions, gradients, filters
✓Light/dark theming via custom properties
✓In-memory state (per session)
✓Keyboard nav & ARIA (built in throughout)
✗External scripts / CDNs (mermaid, D3, React-UMD)
✗Network: fetch / XHR / WebSocket to any host
✗localStorage / IndexedDB (sandbox blocks storage)
✗Remote fonts, stylesheets or images
✗Server-side code or a database

Every ✗ has a workaround shown above: diagrams → hand-built SVG · libraries → inline the logic · data → embed it · fonts → system stack or @font-face data URI. This entire page is one HTML file, no requests after load.
